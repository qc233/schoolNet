## 描述
本项目为中原工学院校园网逆向
能够模拟网页登陆的方式（一键）连接校园网

## 环境
开发环境为 **python3.8.10**

## 安装
~~~bash
pip install -r requirements.txt
~~~

* 使用清华镜像源
    ~~~bash
    pip install -r requirements.txt -i https://pypi.tun.tsinghua.edu.cn/simple
    ~~~
